# Hírbeszéd – képernyőindex

Az összes drótváz megtekinthető az [osszes-oldal.html](osszes-oldal.html) fájlban. A fejlécben a **Világos**, **Sötét** és **Mindkettő** kapcsolóval mind a 33 képernyő teljes témaváltozata megjeleníthető.
